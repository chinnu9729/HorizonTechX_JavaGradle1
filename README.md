# Horizon TechX Java Gradle Project

## Project Title

Java Application using Gradle

---

## Objective

This project demonstrates how to automate the build and execution of a Java application using Gradle and Jenkins. It is developed as part of the Horizon TechX DevOps Internship.

---

## Technologies Used

- Java 21
- Gradle 8.7
- Jenkins
- Git
- GitHub
- AWS EC2 (Ubuntu 24.04)

---

## Prerequisites

- Ubuntu Linux
- Java 17
- Gradle 8.7
- Git
- Jenkins

---

## Installation Steps

### Clone Repository

```bash
git clone https://github.com/chinnu9729/HorizonTechX_JavaGradle1.git
```

### Go to Project

```bash
cd HorizonTechX_JavaGradle1
```

### Build Project

```bash
./gradlew clean build
```

### Run Project

```bash
./gradlew run
```

---

## Build Commands

```bash
./gradlew clean build
```

---

## Run Commands

```bash
./gradlew run
```

---

## Jenkins Integration

1. Installed Jenkins on AWS EC2.
2. Connected GitHub Repository.
3. Created Freestyle Project.
4. Configured Git Repository.
5. Executed Build using:

```bash
chmod +x gradlew
./gradlew clean build
./gradlew run
```

6. Build completed successfully.
